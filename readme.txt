Falcon Dodge!

In this game it is your job, Captain Falcon, to dodge the enemies! You start out with 50 health and get damaged when the enemies attack (are on top of) you. Have no fear though, you can kill them by Falcon Punching them with the B button. Kill all the enemies to win the game!

Controls
Start - Start the game
Up, Down, Left, Right -  Move around with Falcon
B - Falcon Punch
Select - Restart the game at any point
